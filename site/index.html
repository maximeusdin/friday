Friday Research Console

AI research assistant for Cold War archival materials

Build the Application

This is the source directory. To deploy to S3:

npm install

npm run build

The built site will be in the out/ folder.

Configure API URL

Set the backend API URL before building:

NEXT_PUBLIC_API_URL=https://your-api.com/api

Deploy to S3

After building, deploy with:

.\deploy.ps1 -BucketName your-bucket

or

./deploy.sh your-bucket
